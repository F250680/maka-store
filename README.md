# Maka Store

Maka Store es una aplicación de gestión de productos y pedidos desarrollada con **Spring Boot**, diseñada para facilitar la administración de una tienda.

## 🔧 Tecnologías usadas

- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- IntelliJ IDEA
- Maven

## 📦 Funcionalidades

- Gestión de categorías
- Gestión de clientes
- Gestión de pedidos
- Persistencia con base de datos MySQL

## 🚀 Cómo ejecutar

1. Asegúrate de tener una base de datos `maka_store` en MySQL.
2. Configura tu `application.properties` con las credenciales correctas.
3. Ejecuta el proyecto desde IntelliJ con `MakaStoreApplication.java`.
